Este repositório contém o código-fonte e os projetos desenvolvidos durante o Bootcamp Santander 2024, em parceria com a DIO. A formação abrangeu uma ampla gama de tópicos, desde os conceitos básicos de Java até o desenvolvimento de APIs robustas com Spring.

## Resumo

Durante o bootcamp, explorei conceitos fundamentais de Java e programei soluções completas utilizando Spring. Os desafios e projetos ao longo do curso ajudaram a aprimorar minhas habilidades e a construir um portfólio sólido.

---

## Cursos Realizados

### 1. Princípios de Desenvolvimento de Software Colaborativo

- Introdução à Plataforma Java
- Versionamento de Código com Git e GitHub
- Desafio de Projeto: Crie um Portfólio Vencedor
- Contribuindo em um Projeto Open Source no GitHub

### 2. Dominando a Linguagem de Programação Java

- Ambiente de Desenvolvimento Java
- Aprendendo a Sintaxe Java
- Desafio de Projeto: Simulando uma Conta Bancária via Terminal/Console
- Introdução e Estruturas Condicionais com Java
- Estrutura de Repetição em Java
- Tratamento de Exceções em Java
- Debugging Java
- Desafio de Projeto: Criando um Sistema de Validação de Processo Seletivo
- Desafios de Código: Aperfeiçoe sua Lógica e Pensamento Computacional
- Desafios de Código: Condições, Loops e Exceções na Prática com Java

### 3. Programação Orientada a Objetos com Java

- Fundamentos da POO com Java
- Pilares da POO em Java
- Desafio de Projeto: Modelando o iPhone com UML
- Conhecendo Collections Java
- Ganhando Produtividade com Stream API
- Desafio de Projeto: Criando um Banco Digital com Java e POO
- Desafio de Projeto: Abstraindo um Bootcamp Usando POO em Java
- Desafio de Projeto: Orientação a Objetos na Prática com Java

### 4. Teste e Gerenciamento de Dependências em Projetos Java

- Gerenciamento de Dependências e Build em Java com Maven
- Introdução a Testes de Software
- Testes Unitários com JUnit
- Desenvolvimento de Testes Utilizando Mockito

### 5. Banco de Dados SQL e NoSQL para Desenvolvedores Back-end

- Introdução a Bancos de Dados Relacionais (SQL)
- Introdução ao MongoDB e Bancos de Dados NoSQL

### 6. Ganhando Produtividade no Java com Spring Framework

- Introdução ao Spring Framework com Spring Boot
- Criando uma API REST Documentada com Spring Web e Swagger
- Adicionando Segurança a uma API REST com Spring Security
- Desafio de Projeto: Design Patterns com Java - Dos Clássicos (GoF) ao Spring Framework
- Desafio de Projeto: Publicando sua API REST na Nuvem com Spring Boot 3, Java 17 e Railway

### 7. Soft Skills na Era Ágil: Conectando Pessoas e Oportunidades

- Trabalhando em Equipes Ágeis
- Deixando seu LinkedIn Atraente
- Transformando seu Conhecimento em Artigos Técnicos
- Preparação para Entrevistas
